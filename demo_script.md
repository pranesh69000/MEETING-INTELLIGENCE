# Demo Script: Recruiter Interview (2 Minutes)

**Roleplay Instructions:**
*   **PAUSES**: Wait for the full [2 SECONDS] where marked. This is crucial for the AI to label "Speaker A" aka Recruiter.
*   **INTONATION**: Ask questions clearly (?) so the system detects the pitch change.

---

**Speaker A (Recruiter):**
"Hi, thanks for joining. I looked at your resume and found this 'Meeting Intelligence' project very interesting. Can you tell me what it does?"
*[PAUSE - 2 SECONDS]*

**Speaker B (Developer/Candidate):**
"Sure. It is a local-first desktop application that records meetings, transcribes audio offline, and generates summaries automatically."
*[PAUSE - 2 SECONDS]*

**Speaker A:**
"That sounds useful. What technology stack did you use to build it?"
*[PAUSE - 2 SECONDS]*

**Speaker B:**
"I used React for the frontend and Python FastAPI for the backend. For the AI part, I integrated OpenAI's Whisper model running locally on the device."
*[PAUSE - 2 SECONDS]*

**Speaker A:**
"Impressive. Does it handle privacy securely? Many clients are worried about cloud data."
*[PAUSE - 2 SECONDS]*

**Speaker B:**
"Yes, exacty. Since Whisper runs offline, no audio ever leaves the user's computer. Only the final text notes are synced to Google Drive."
*[PAUSE - 2 SECONDS]*

**Speaker A:**
"Okay, that is a strong selling point. Can you send me the GitHub repository link?"
*[PAUSE - 2 SECONDS]*

**Speaker B:**
"I will email the GitHub link to you by this evening."
*[PAUSE - 2 SECONDS]*

**Speaker A:**
"Great. Also, we should schedule a technical round for next week. Are you available on Tuesday?"
*[PAUSE - 2 SECONDS]*

**Speaker B:**
"Yes, Tuesday works for me. I will check my calendar and confirm the time."
*[PAUSE - 2 SECONDS]*

**Speaker A:**
"Perfect. I will send the calendar invite shortly. looking forward to it."
